## Loading the Extension for usage

Ensure you're using manifest_version: 3 and make sure your files are correctly placed in the directory.

```shell
.
├── background.js
├── content.js
├── icons
│   ├── icon128.png
│   ├── icon16.png
│   └── icon48.png
├── image.png
├── manifest.json
├── popup.html
└── popup.js
```
